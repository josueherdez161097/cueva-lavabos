# Cueva Web Base

Base inicial en **Next.js + TypeScript + Tailwind CSS + Framer Motion** para la marca **Cueva**.

## Incluye

- Home con hero premium
- Catálogo de productos
- Página dinámica por producto
- Tema centralizado en `src/lib/theme.ts`
- Componentes modulares
- Estructura lista para crecer a CMS, ecommerce o cotización por WhatsApp

## Instalación

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`

## Estructura

```bash
src/
  app/
  components/
  data/
  lib/
```

## Personalización rápida

### Paleta
Edita variables en:

- `src/app/globals.css`
- `src/lib/theme.ts`

### Productos
Edita el arreglo en:

- `src/data/products.ts`

## Siguientes pasos sugeridos

1. Reemplazar imágenes de Unsplash por fotos reales de Cueva.
2. Conectar catálogo a CMS o base de datos.
3. Añadir formulario de contacto real y analytics.
4. Integrar SEO avanzado y esquema de producto.
5. Agregar carrito o flujo de cotización.
